# Fibarium Protocol

Fibarium protocol is a web3 identity management protocol on the blockchain network

## Install

### Install dependencies

```bash
npm install
```

### Compile Contracts

```bash
npx hardhat compile
```

---

## Testing

### Run all tests:

```bash
npx hardhat test
```


## Deployment

### Deployment smart contracts on-chain:

```bash
npx hardhat run --network <network> ./scripts/deploy.js 
```
